
## Unidades Absolutas

*px* - pixels

*pt* - definição de fonte

*cm* - centímetro 


## Unidades Relativas

*em* - baseia-se sempre no tamanho do elemento pai

*rem* - baseia-se no tamanho da estrutura do html

*%* - porcentagem

### Unidades de medida baseado em espaço visível

*vh* - view heigh 

*vw* - view width
